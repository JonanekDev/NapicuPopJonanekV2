# PopJonanek 2 - Web
## Instalace
1. Naklonování repozitáře
    ```sh
    git clone https://github.com/Numax-cz/NapicuPopJonanekV2.git
   ```
2. Nainstalování balíčků (web)
   ```sh
   npm install
   ```
## Spuštění 
1. Spuštění aplikace na localhostu
   ```sh
   npm run start
   ```
2. Buildnutí aplikace pro finální použití
   ```sh
   npm run build
   ```
